再见, 2017

发表于 2017/12/31 | 分类于 Essay | 阅读次数

今天，是2017年的最后一天
再过不到三个小时，就是2018了
过去这一年，说起来，生活好像也没有多大变化

首先，说说学习
马上，大学生涯就只剩下三分之一了
原本看样子，大二下学期的时候成绩有些起色
但最后发现，一整年下来
成绩毫无变化

大三也只剩下一半了
有的路不得不走，所以还是要走
比如考研也许就是其中之一

生活上，2017年印象最深刻的大概就是感冒必发烧吧
其中比较严重的两次还不得不进医院输液
一次是四月份，刚过完清明，到医院输了三天液
一次就是上周，到医院输了四天液

过去的一年好像也熬过不少夜
大概还通过四次宵
不过这真的是一个坏习惯

作为一个计系学生
过去这一年应该也写了不少代码
目前最熟练的语言还是 C 吧
虽然不得不写了很多的 Verilog，但真的不感兴趣
Python 是一门很强大的语言，于是便成为了我第二喜爱的语言
这学期因为数据库实验学了一下世界上最好的语言，也写了一点东西
不过目前真的只懂一些基础语法
JS/Java/C++/HTML/CSS 大概是可以略过了
实践是学习一门语言最快的方式

2017 年最大的一个变化大概就是生命中多了一个人
多了一个让我朝思暮想的人
嘻嘻，你们懂的

每次写东西前好像都感觉有很多东西要写
但真正开始写的时候却发现什么也写不出来
最后只能胡说八道一些
大概就这些吧

新的一年
希望自己还有亲人、爱人都能有一个好身体
希望成绩能有所长进
希望考研顺利
希望和女朋友能少吵几次架
舍不得看到她哭
还有一个最大的愿望就留在心里当作秘密吧
因为我害怕说出来就不能实现了
