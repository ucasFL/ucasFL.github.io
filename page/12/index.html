前言

因对 Linux 命令行的喜爱，故入手《Linux 命令行大全》一书，打算从头至尾阅读、学习一遍，并将书中每一章节的命令集中整理如下。

第一部分：学习 shell

Chapter 1 学习 shell

  • $ date : 显示当前系统的时间和日期
  • $ cal : 显示日历
  • $ df : 查看磁盘驱动器可用空间
  • $ free : 显示可用内存
  • $ exit : 关闭终端窗口

Chapter 2 导航

  • $ pwd : 显示当前工作目录
  • $ ls : 列出当前工作目录的文件和目录
  • $ cd : 更改当前工作目录（路径名）
  • 绝对路径名： 从根目录开始，通向目标目录
  • 相对路径名： 从工作目录开始，通向目标目录
