Sudo Apt Update失败的解决办法

发表于 2016/12/19 | 分类于 Linux | 阅读次数

在 ubuntu 系统上，当我们通过 apt 软件包安装新的软件时，都要先进行软件包更新：

~$ sudo apt update

但有时候，会出现 error， 如下图所示：

这时候，我们只需要把 /var/lib/apt/lists 目录中的文件全部删除，然后再重新更新就好了：

~$ rm -r /var/lib/apt/lists
~$ sudo apt update

OK ~ 问题解决
